Visualizing U.S. Aircraft Incidents
A First Glance, the Data as a Whole
We first plot every U.S.-related airplane incident that has an available location. Using this kind of view, we can easily gather specific information about each incident by hovering. However, because the circles have overlap, this kind of view makes it difficult to see where incidents are concentrated.
Heat Maps
By changing these incident circles into heat circles, we can see where incidents are most concentrated. However, because of the overlap, it is no longer useful to provide an information tooltip.
